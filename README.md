# StarkNews

მოქალაქეობრივი პრესის ვებგვერდი — ახალი ამბების პორტალი, სადაც ნებისმიერ დარეგისტრირებულ
მომხმარებელს შეუძლია დაამატოს სიახლე (სათაური, ტექსტი, ფოტოს ბმული), რომელიც ავტომატურად
გამოჩნდება მთავარ გვერდზე სიახლეების ლენტაში. მომხმარებლებს ასევე შეუძლიათ სიახლეების
მოწონება ("ლაიქი"), რაც განსაზღვრავს რომელი ამბავია უფრო აქტუალური.

## გამოყენებული ტექნოლოგიები
- Python / Flask
- Flask-SQLAlchemy (SQLite ბაზა)
- Flask-Login (რეგისტრაცია/ავტორიზაცია)
- Jinja2 (შაბლონები)
- Bootstrap 5

## პროექტის სტრუქტურა
```
StarkNews/
├── app.py            # აპლიკაციის შესვლის წერტილი და routes-ის რეგისტრაცია
├── ext.py            # db და login_manager ინსტანციები
├── models.py         # User, Article, Like მოდელები
├── routes.py         # ყველა view ფუნქცია
├── requirements.txt
├── static/
│   └── style.css
└── templates/
    ├── base.html, header.html, header_login.html, header_authenticated.html, footer.html
    ├── index.html          # მთავარი გვერდი / სიახლეების ლენტა
    ├── login.html / register.html
    ├── add_news.html       # სიახლის დამატების ფორმა
    ├── edit_news.html      # სიახლის რედაქტირება
    ├── news_detail.html    # სრული სიახლე + მოწონების ღილაკი
    ├── my_news.html        # ჩემი დამატებული სიახლეები
    └── news_card.html      # ერთი სიახლის ბარათი (partial)
```

## გაშვება ლოკალურად
```
pip install -r requirements.txt
python app.py
```
სერვერი გაეშვება `http://127.0.0.1:5000`-ზე. ბაზა (`StarkNewsDB.db`) იქმნება ავტომატურად პირველივე
გაშვებისას.

## ფუნქციონალი
- რეგისტრაცია და ავტორიზაცია (Flask-Login)
- სიახლის დამატება, რედაქტირება და წაშლა (მხოლოდ ავტორს შეუძლია)
- მთავარ გვერდზე ყველა სიახლის ავტომატური ასახვა
- სიახლეების მოწონების ("ლაიქის") სისტემა
- პირადი კაბინეტი — "ჩემი სიახლეები"
